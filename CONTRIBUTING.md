# Contributing

This public repository contains app code, documentation, templates, and synthetic examples for the ARG Risk & Abundance-Informed Index Tool.

## Main rule

Do not add private project data to this repository. Use synthetic examples for public tests and documentation.

## Code changes

Before proposing code changes:

1. Run the app locally with the demo table.
2. Check lookup, upload, index calculation, and downloads.
3. Run the test suite.
4. Check that only public-safe files are staged.

## Documentation changes

Documentation should preserve the main interpretation points:

- final scores are relative prioritization metrics,
- the abundance-informed index depends on abundance units,
- unmatched ARGs require review,
- the tool is for research prioritization.

## Issue reports

When reporting an issue, include:

- what you were trying to do,
- the app tab involved,
- the error message,
- whether demo mode or private-data mode was used,
- a small synthetic example if data are needed.

Do not paste private data into public issues.
