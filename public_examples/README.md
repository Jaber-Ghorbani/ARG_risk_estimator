# Public Example Files

This folder contains public-safe templates and synthetic example files for the ARG Risk & Abundance-Informed Index Tool.

## Files

| File | Purpose |
|---|---|
| `minimal_template.csv` | Required upload columns only. |
| `extended_template.csv` | Required columns plus optional metadata columns. |
| `example_dataset.csv` | Synthetic dataset for app demonstration. |
| `example_output.csv` | Synthetic example output for documentation and testing. |
| `demo_score_table.csv` | Tiny synthetic score table used when the private score table is absent. |

## Important note

The files in this folder are not the full ARG score database. They are small public examples intended to help users understand formatting and app behavior.

The full score table should remain outside the public repository.
